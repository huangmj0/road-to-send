How it works

Score the work that gets you there.

Each person can earn up to 16 activity points per week, plus a +2 balanced-week bonus and up to two 1–3 point daily bounties: 24 points maximum. Longer purposeful bouldering sessions earn modestly more without letting available free time dominate.

💪2 ptsPull session15–30 clean, controlled reps · up to 3×/week
🛡️1 ptCore, prehab or mobility10+ purposeful minutes · activity limits apply
🎯+1–3Daily bountyScaled by difficulty · first two each week score

Four ways to win

Points champion rewards commitment. Most improved compares each climber’s final send pyramid with their own Week 1 baseline. Bounty hunter goes to the climber who logs the most completed bounties. Best teammate is decided by the crew. No gender categories or absolute-grade points are used.

Shared logging setup

One person completes this once (~10 minutes):

  1. Create a Google Sheet. Open Extensions → Apps Script.
  2. Paste the script below, replacing any existing code.
  3. Choose Deploy → New deployment → Web app. Execute as you; allow access to anyone with the link.
  4. Paste the deployment URL and challenge settings below. API v6 creates centralized Activities, Settings, Participants, and Benchmarks tabs.

Upgrading an earlier deployment? The first v5 request archived legacy activity data; upgrading from v5 to v6 preserves existing entries and adds the challenge window. Paste the new script over the old one and choose Deploy → Manage deployments → Edit (✎) → Version: New version → Deploy. The web-app URL stays the same.

After setup, you can edit the Settings and Participants tabs directly. Keys ignore capitalization, spaces, underscores, and hyphens; dates accept common Sheet formats and the goal must be a whole number from 50 to 10,000. Security note: anyone with the crew link can submit entries or change setup, so keep it within the group.
