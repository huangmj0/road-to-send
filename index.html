How it works

Score the work that gets you there.

Each person can earn up to 14 points per week. The cap keeps this about sustainable preparation—not whoever can manufacture the most workouts.

🧗3 ptsClimbing session45+ purposeful minutes
💪2 ptsPull session15–30 quality reps
🛡️2 ptsPrehab10 min shoulders, elbows, wrists
🧘1 ptMobility or rest10 min—or intentional recovery

Three ways to win

Points champion rewards consistency. Most improved rewards progress from the Week 1 baseline. Best teammate is decided by the crew. Raw strength never gets the only trophy.

Make the log collective

Recommended: host this single HTML file on GitHub Pages, then connect it to a free Google Sheet through Apps Script. The generated crew link carries the connection and challenge settings automatically; everyone logs through the same page and sees the same leaderboard.

Without a shared endpoint, entries stay in this browser only. That mode is useful for testing, but it is not collective.

Shared logging setup

One person completes this once (~10 minutes):

  1. Create a Google Sheet. Open Extensions → Apps Script.
  2. Paste the script below, replacing any existing code.
  3. Choose Deploy → New deployment → Web app. Execute as you; allow access to anyone with the link.
  4. Paste the deployment URL and challenge settings below. Save, then copy the generated crew link.
Security note: anyone with the crew link can submit entries. That is usually reasonable for a friends-only challenge; don’t use the Sheet for sensitive information.
