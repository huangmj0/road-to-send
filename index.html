How it works

Score the work that gets you there.

Each person can earn up to 14 points per week, plus a +2 balanced-week bonus for mixing climbing, strength and care work (16 max). Every activity also has a weekly limit, so the cap keeps this about sustainable preparation—not whoever can manufacture the most workouts.

🧗3 ptsClimbing session45+ purposeful minutes · up to 4×/week
💪2 ptsPull session15–30 quality reps · up to 3×/week
🛡️2 ptsPrehab10 min shoulders, elbows, wrists · up to 4×/week
🧘1 ptMobility or rest10 min—or intentional recovery · mobility 4×, rest 2×/week

Three ways to win

Points champion rewards consistency. Most improved rewards progress from the Week 1 baseline. Best teammate is decided by the crew. Raw strength never gets the only trophy.

Shared logging setup

One person completes this once (~10 minutes):

  1. Create a Google Sheet. Open Extensions → Apps Script.
  2. Paste the script below, replacing any existing code.
  3. Choose Deploy → New deployment → Web app. Execute as you; allow access to anyone with the link.
  4. Paste the deployment URL and challenge settings below. Saving creates centralized Settings and Participants tabs in the Sheet.

Already deployed an earlier version of the script? Paste the new script over the old one, then choose Deploy → Manage deployments → Edit (✎) → Version: New version → Deploy. The web-app URL stays the same, so nobody needs a new crew link.

After setup, you can edit the Settings and Participants tabs directly. Keys ignore capitalization, spaces, underscores, and hyphens; dates accept common Sheet formats and the goal must be a whole number from 50 to 10,000. Security note: anyone with the crew link can submit entries or change setup, so keep it within the group.
